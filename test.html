This page is a series of informal tests I developed while building text-decoration-skip-ink, text-underline-offset, and text-decoration-thickness. The animation moves the underline up and down using text-underline offset, changes the thickness with text-decoration-thickness, and avoids skips ink as it moves through the text.

I used these tests to avoid regressions, ensure each feature worked together correctly, discover edge cases for formal tests, and communicate with my manager and team members about the state of my project.
